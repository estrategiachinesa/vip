# **App Name**: Trading Ascent

## Core Features:

- Landing Page Rendering: Display a concise, conversion-optimized landing page.
- Urgency Timer: Implement a countdown timer that adds urgency.
- CTA Buttons: Implement prominent call-to-action buttons to drive conversions.
- Testimonial Highlighting: AI tool that highlights the most impactful parts of testimonials to make them visually stand out.
- Custom Video Player: Integrate a custom YouTube video player (videoId: 1iAgu5fAod0) that displays a purchase CTA button upon completion, as requested by the user.

- Primary color: Saturated red (#D42323) to evoke passion and energy in the user, matching the user request.

- Accent color: Gold (#FFD700) to suggest the trading themes, matching the user request.
- Headline font: 'Poppins' (sans-serif) for headlines, as requested by the user. Note: currently only Google Fonts are supported.
- Body font: 'PT Sans' (sans-serif) for body text, as requested by the user. Note: currently only Google Fonts are supported.
- Incorporate clean and minimal icons from lucide-react, subtly referencing both trading and East Asian culture, matching the user request.
- Maintain a clean and structured layout with clear sections and intuitive navigation to facilitate user engagement, matching the user request.
- Introduce subtle animations such as fade effects and smooth transitions to enhance the user experience without distracting from the main message, as per the user request.
